translate()関数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(480, 120)

　　 ｝

　　

　　 void draw(){

　　 　translate(mouseX, mouseY);

　　 　rect(0, 0, 30, 30);

　　 　translate(35, 10);

　　 　rect(0, 0, 15, 15);

　　 }　　

説明文

　基準となる座標を指定することができる関数。

　基本的な座標の基準は画面左上の（0，0）となる。

　translate(x, y)のx座標とy座標を指定することで基準の座標をずらすことができる。

　例では、mouseX,Y変数を用いることでマウスカーソルの位置を基準点（0，0）に変更している。

　その後、2つ目のtranslate()でマウスの位置を（0，0）として座標が（35，10）の位置に2つ目の四角形ができるようになっている。
